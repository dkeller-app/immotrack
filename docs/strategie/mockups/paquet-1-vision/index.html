② Bien vs Locataires — clarification
Aujourd'hui : la fiche d'un bien mélange infos du mur (surface, DPE) avec infos du locataire (nom, mail). C'est confus.
Cible : deux onglets nets. Bien = je gère le mur. Locataires = je gère la personne. Le bail apparaît dans les deux mais sous des angles différents.
💡
La règle pour ne pas se tromper

Quand tu te demandes "où je clique ?" : Bien pour tout ce qui est immobilier/physique/légal, Locataires pour tout ce qui est relationnel/contractuel.
