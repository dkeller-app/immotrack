Assurances — la PNO se gère dans la fiche immeuble

Cible validée : la PNO / multirisque (qui couvre TOUT l'immeuble) vit dans la fiche immeuble, comme les charges communes. La fiche logement ne garde que la MRH du locataire — plus aucun affichage de PNO héritée (zéro valeur ajoutée). Ce mockup réutilise le vrai css/main.css : le rendu est identique à l'app. Redimensionne la fenêtre pour voir le comportement PC / tablette / téléphone.
