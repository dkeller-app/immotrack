Mon raisonnement sur les 6 Quick Access

Le but : 2 clics max pour accéder aux 6 actions les plus fréquentes d'un bailleur. Chaque tile a un mini-stat live (ex: "21/22 payés") pour pousser à cliquer + un badge rouge/orange si action requise.

Alternatives écartées (pour info)

  • 🛡 Assurances : utile mais ~3-4 actions par an seulement, pas dans le top 6 quotidien
  • 📊 Révision IRL : ~1 fois par an par lot, mieux à côté de "Loyers"
  • 📋 Baux : consultation moins fréquente que Logements/Locataires
  • 📊 Export / 2044 : 1 fois par an, à mettre dans Paramètres
  • 🛠 Équipements : consultation rare, accessible via fiche Logement

Variante mode Solo (sans dashboard)

En mode Solo, la tile "Tableau de bord" disparaît → reste 5 tiles. Pour garder un grid esthétique, on peut :

  • Option 1 (recommandée) : grid 3×2 → 5 tiles + 1 case vide (pas grave, ou y mettre une CTA "💎 Passer en Premium")
  • Option 2 : grid 3+2 (3 cards en haut, 2 plus larges en bas)
  • Option 3 : grid 5 cols × 1 row (tiles plus petites)
