🤖 Règles de remplissage automatique
À l'import bancaire, si le libellé contient le motif, les champs sélectionnés sont pré-remplis. Les règles s'appliquent dans l'ordre.
💡 Exemple : motif CREDIT AGRICOLE 12345 → Catégorie = Prêt, Logement = F-001
🎨 Couleurs de l'interface
Personnalisez les couleurs d'ImmoTrack. S'applique uniquement à l'interface — les documents générés (bail, EDL, quittances) restent en noir/blanc.
⚠ Le thème est sauvegardé dans votre navigateur. Il ne s'applique pas aux autres appareils sauf si vous exportez/importez la base de données.

ID du dossier Drive utilisé par défaut pour synchroniser les photos des EDL.
