💧 Eau chaude sanitaire
Modèle de données : ces cases vivent désormais sur log.chauffage (le bien), pas sur le bail. Le bail les lit via _readLogForBail() → source unique. Le wizard bail « Le bien » reste en lecture seule (déjà le cas). Les anciens champs bail.chauffFioul/Bois/… sont supprimés ; la migration au boot recopie les valeurs existantes des baux vers le bien.
