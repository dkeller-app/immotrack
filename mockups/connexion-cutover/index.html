Ce qui disparaît (sans interface)

Débranché / retiré

✕

✕
Réglages Drive + export/import Drive + « PDF → Drive ».
✕
Bandeau bleu (mode cloud / « un autre appareil a modifié »).
✕
Session persistante : mot de passe redemandé à chaque ouverture.

Conservé (intact)

✓
Tes photos & fichiers sur Google Drive : non supprimés. L'app cesse seulement de s'y connecter.
✓
Tout le reste de l'app sur Supabase (biens, mouvements, baux, EDL, partage SCI…).
✓
Les ancres d'authentification (formulaire, flux invitation) — refonte visuelle seulement.
